Verified the mechanics. Here's the precise situation and what the handoff actually is.     
                                                                                          
  Why converge is broken today (withSimpleCache.ts:306):                                     
  if (!(conditionOption && error instanceof SimpleCacheConditionError))                      
  instanceof checks prototype identity. Our local SimpleCacheConditionError is a different   
  constructor from the wrapper's, so our throw fails that check → the wrapper re-throws      
  instead of converging. No amount of same-naming fixes instanceof.                          
                                                                                             
  One useful thing I found: both classes extend ConstraintError from helpful-errors, and     
  HelpfulError builds its message prefix from new.target.name — so both throw ✋             
  SimpleCacheConditionError: … and both share error.constructor.name === 
  'SimpleCacheConditionError'. That gives a structural discriminator that works across       
  package boundaries, which instanceof cannot.                                               

  So the handoff = a change to with-simple-cache's converge guard (nothing to hand off from  
  our repo). Three ways, cheapest first:
                                                                                             
  Option 1 — structural name check (our side stays "just extends ConstraintError")           
  // withSimpleCache.ts + withSimpleCacheAsync.ts, the converge guard
  if (!(conditionOption &&                                                                   
        error instanceof ConstraintError &&                                                  
        error.constructor?.name === 'SimpleCacheConditionError'))
  Recognizes any backend's SimpleCacheConditionError, zero extra work on our side. Tradeoff: 
  relies on the class name surviving (fine for node libs; risky only under aggressive        
  minification).
