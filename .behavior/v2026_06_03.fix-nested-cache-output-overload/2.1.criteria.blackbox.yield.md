# blackbox criteria: meta include/exclude

## usecase.1 = include cache metadata

```
given(withSimpleCacheAsync wrapped function with meta: 'include')
  when(function is called)
    then(returns { output, cached } object)
      sothat(caller can access both result and cache metadata)
    then(output contains the function's return value)
      sothat(caller can use result as before)
    then(cached.uri is present when cache supports uri)
      sothat(caller knows where data was persisted)
    then(cached.uri is undefined when cache does not support uri)
      sothat(caller handles gracefully without error)

given(withSimpleCacheAsync wrapped function with meta: 'include')
  when(function is called and result is served from cache)
    then(returns { output, cached } with same shape as cache miss)
      sothat(caller code works identically for hit or miss)
    then(cached.uri reflects where data was retrieved from)
      sothat(caller has correct location regardless of hit/miss)

given(withSimpleCacheAsync wrapped function with meta: 'include')
  when(function returns undefined)
    then(output is undefined in { output, cached })
      sothat(undefined results are preserved correctly)
```

## usecase.2 = exclude cache metadata (default)

```
given(withSimpleCacheAsync wrapped function with meta: 'exclude')
  when(function is called)
    then(returns the function's return value directly)
      sothat(extant behavior is preserved)
    then(return type matches original function signature)
      sothat(no breaks for extant callers)

given(withSimpleCacheAsync wrapped function with no meta option)
  when(function is called)
    then(returns the function's return value directly)
      sothat(default behavior is backwards compatible)
```

## usecase.3 = type safety

```
given(typescript user with meta: 'include')
  when(destructures return value)
    then(compiler knows { output, cached } shape)
      sothat(typos are caught at compile time)
    then(cached.uri type is string | undefined)
      sothat(caller must handle optional uri)

given(typescript user with meta: 'exclude')
  when(accesses return value)
    then(compiler knows original return type)
      sothat(no type changes for extant code)
    then(cannot destructure { output, cached })
      sothat(compiler catches incorrect usage)
```

## usecase.4 = dynamic cache extraction with meta

```
given(withSimpleCacheAsync with cache extraction function and meta: 'include')
  when(function is called with context that has cache)
    then(cache is extracted from args at runtime)
      sothat(context-based caches work with meta option)
    then(cached.uri reflects extracted cache's uri)
      sothat(dynamic cache uri is correctly exposed)
```
