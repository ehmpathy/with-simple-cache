# playtest: meta option for cache metadata access

## prerequisites

- node.js 18+ installed
- pnpm installed
- repo cloned and dependencies installed (`pnpm install`)

## sandbox

all file operations in this playtest target `.temp/` in the repo root.

## happy paths

### 1. verify meta: include returns { output, cached } with uri

create test file:

```bash
mkdir -p .temp
cat > .temp/playtest-meta-include.ts << 'EOF'
import { withSimpleCacheAsync } from '../src/index';
import { createCache } from 'simple-in-memory-cache';

// create cache with uri method
const cache = {
  ...createCache(),
  uri: (key: string) => `s3://test-bucket/${key}`,
};

const fetchData = withSimpleCacheAsync(
  async (input: { searchId: string }) => {
    console.log('  [api call]');
    return { message: 'hello world' };
  },
  { cache, meta: 'include' },
);

(async () => {
  console.log('1. first call (cache miss):');
  const result1 = await fetchData({ searchId: 'abc-123' });
  console.log('  result:', JSON.stringify(result1, null, 2));

  console.log('\n2. second call (cache hit):');
  const result2 = await fetchData({ searchId: 'abc-123' });
  console.log('  result:', JSON.stringify(result2, null, 2));
})();
EOF
```

run:

```bash
npx tsx .temp/playtest-meta-include.ts
```

**expected output:**
```
1. first call (cache miss):
  [api call]
  result: {
  "output": {
    "message": "hello world"
  },
  "cached": {
    "uri": "s3://test-bucket/{\"searchId\":\"abc-123\"}"
  }
}

2. second call (cache hit):
  result: {
  "output": {
    "message": "hello world"
  },
  "cached": {
    "uri": "s3://test-bucket/{\"searchId\":\"abc-123\"}"
  }
}
```

**pass if:**
- result has `output` and `cached.uri` properties
- api call happens only once (cache hit on second call)
- uri contains `s3://test-bucket/` with serialized key

**fail if:**
- result is raw value without wrapper
- api call happens twice
- error is thrown

### 5. verify extraction function with meta: include (primary use case)

this is the primary use case from the wish: extract cache from context parameter.

create test file:

```bash
cat > .temp/playtest-extraction-fn.ts << 'EOF'
import type { SimpleCache, WithCacheUri } from '../src/domain.objects/SimpleCache';
import { withSimpleCacheAsync } from '../src/index';

// define extraction function OUTSIDE to preserve return type
// note: inline extraction functions lose return type due to TypeScript
//       constraint inference (widens to `any` from WithSimpleCacheChoice)
const extractCache = ({
  fromInput,
}: {
  fromInput: [
    { searchId: string },
    { cache: WithCacheUri<SimpleCache<string>> },
  ];
}) => fromInput[1].cache;

const runPlay = async (
  input: { searchId: string },
  context: { cache: WithCacheUri<SimpleCache<string>> },
): Promise<string> => {
  console.log('  [api call]');
  return 'search results csv';
};

const cachedRunPlay = withSimpleCacheAsync(runPlay, {
  cache: extractCache,
  meta: 'include',
});

(async () => {
  // create cache with uri method (e.g., S3 file cache)
  const s3Cache: WithCacheUri<SimpleCache<string>> = {
    get: async () => undefined,
    set: async () => {},
    uri: (key: string) => `s3://my-bucket/artifacts/${key}`,
  };

  console.log('1. first call (cache miss):');
  const result1 = await cachedRunPlay({ searchId: 'abc-123' }, { cache: s3Cache });
  console.log('  output:', result1.output);
  console.log('  cached.uri:', result1.cached.uri);

  console.log('\n2. second call (cache hit):');
  const result2 = await cachedRunPlay({ searchId: 'abc-123' }, { cache: s3Cache });
  console.log('  output:', result2.output);
  console.log('  cached.uri:', result2.cached.uri);
})();
EOF
```

run:

```bash
npx tsx .temp/playtest-extraction-fn.ts
```

**expected output:**
```
1. first call (cache miss):
  [api call]
  output: search results csv
  cached.uri: s3://my-bucket/artifacts/{"searchId":"abc-123"}

2. second call (cache hit):
  output: search results csv
  cached.uri: s3://my-bucket/artifacts/{"searchId":"abc-123"}
```

**pass if:**
- result has `output` and `cached.uri` properties
- api call happens only once (cache hit on second call)
- uri contains `s3://my-bucket/artifacts/` with serialized key
- cache extracted from context parameter works correctly

**fail if:**
- TypeScript error on `meta: 'include'`
- result lacks `output` or `cached.uri`
- api call happens twice

## edge paths

### 6. verify runtime guard for meta: include without uri method

create test file:

```bash
cat > .temp/playtest-runtime-guard.ts << 'EOF'
import { withSimpleCacheAsync } from '../src/index';
import { createCache } from 'simple-in-memory-cache';

// cache WITHOUT uri method
const cache = createCache();

const fetchData = withSimpleCacheAsync(
  async (input: { id: string }) => {
    return 42;
  },
  { cache, meta: 'include' as any },  // bypass type check
);

(async () => {
  try {
    await fetchData({ id: 'test-123' });
    console.log('ERROR: should have thrown');
  } catch (error: any) {
    console.log('caught error:', error.message);
    console.log('error type:', error.constructor.name);
  }
})();
EOF
```

run:

```bash
npx tsx .temp/playtest-runtime-guard.ts
```

**expected output:**
```
caught error: meta: 'include' requires cache with uri method
error type: BadRequestError
```

**pass if:**
- BadRequestError is thrown
- error message explains the constraint

**fail if:**
- no error is thrown
- different error type or message
