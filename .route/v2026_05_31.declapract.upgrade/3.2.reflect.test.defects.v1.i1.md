## practice bug: dpdm yq syntax incompatible

**what**: `test:lint:cycles` yq command uses python-yq/jq syntax but mikefarah/yq v4 has different syntax

**how**: yq command `if length > 0 then join("|") else "^$" end` fails silently on mikefarah/yq v4, passes empty exclude pattern to dpdm

**why**: template assumes python-yq/jq syntax but github actions ubuntu-24.04 has mikefarah/yq v4.53.2

**fix**: changed to mikefarah/yq v4 syntax: `.exclude | join("|") // "^$"`

**citation**: `gh search code --repo ehmpathy/declapract-typescript-ehmpathy "dpdm"`

**feedback**: file issue in ehmpathy/declapract-typescript-ehmpathy to use yq v4 compatible syntax
